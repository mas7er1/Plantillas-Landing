TishKiteSurfingHTML Documentation

Introduction

This documentation will guide you through the process of setting up and customizing the TishKiteSurfingHTML template.

Setup

  1. Download the TishKiteSurfingHTML template package.
  2. Copy the css, fonts, and js folders to your website's root directory.

Meta Tags

Include and update the following meta tags in the head section of your HTML:


<meta http-equiv="content-type" content="text/html; charset=utf-8">
  <meta http-equiv="x-ua-compatible" content="IE=edge">
  <meta name="author" content="Tishonator">

CSS Files

Include the following CSS files in the head section of your HTML:


<link rel="stylesheet" href="./css/style.css">

  <!-- Font Icons -->
  <link rel="stylesheet" href="./css/font-icons.css">
  <link rel="stylesheet" href="./css/et-line.css">

  <!-- Plugins/Components CSS -->
  <link rel="stylesheet" href="./css/swiper.css">
  <link rel="stylesheet" href="./css/bs-switches.css">

  <meta name="viewport" content="width=device-width, initial-scale=1">

JS Files

Include the following JS files in the body (above <body> tag) of your HTML:


  <script src="./js/plugins.min.js"></script>
  <script src="./js/functions.bundle.js"></script>

JavaScript Documentation

Loading Animation

The script starts by displaying a loading animation. This is achieved by setting the display property of the element with the ID "loadingAnimation" to "block".

document.getElementById("loadingAnimation").style.display = "block";

Fetching Data

The fetchData() function is a dummy function that simulates fetching data from a server. It uses a timeout of 1.5 seconds and then logs "website data fetched" to the console.

function fetchData() { return new Promise((resolve) => { setTimeout(() => { console.log("website data fetched"); resolve(); }, 1500); }); }

Swiper Slider Initialization

After fetching data, the script initializes the Swiper slider with various settings.

var swiper = new Swiper('.swiper-container', { direction: 'horizontal', loop: true, ... });

DOM Content Loaded

The script contains several functionalities that are executed once the DOM content is fully loaded:

  • Adjusting body padding based on the height of a fixed-top navbar.
    • const navbarHeight = navbarElement.offsetHeight; document.body.style.paddingTop = `${navbarHeight}px`;
  • Adding flip effects to elements with the class "sign".
    • sign.addEventListener('click', function() { if (this.classList.contains('flipped')) { this.classList.remove('flipped'); } else { this.classList.add('flipped'); } });
  • Changing the navbar style based on the scroll position.
    • if (window.pageYOffset > 0) { navbar.classList.add("scrolled"); } else { navbar.classList.remove("scrolled"); }
  • Scroll-to-top functionality for an element with the ID "scrollToTop".
    • window.scrollTo({ top: 0, behavior: "smooth" });
  • Tab functionality for daily/weekly/monthly tabs.
    • let targetContent = document.querySelector(tab.getAttribute('href')); targetContent.classList.add('show', 'active');

Adoption and Modification

To adopt or modify this script for your project:

  1. Ensure that the HTML elements referenced by their IDs or classes in the script exist in your HTML file.
  2. For functionalities that you don't need, simply remove or comment out the corresponding code sections.
  3. Adjust the settings, such as the Swiper slider settings or the timeout duration in the fetchData() function, to fit your project's requirements.
  4. If you have different IDs or class names in your HTML, make sure to update the script to reference the correct elements.
